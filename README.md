# Game Foundation

A Godot 4.5 project with a structure-first architecture.

## Structure

- `src/core/` — Foundation layer (bootstrapping, routing, save/load)
- `src/features/` — Isolated feature modules
- `src/shared/` — Reusable building blocks
- `src/systems/` — Long-lived runtime systems
- `src/ui/` — Global UI shell
- `src/content/` — Data-only content
- `src/autoloads/` — Singleton services
